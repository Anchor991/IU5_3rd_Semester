该课程只有一个老师，有演讲课和研讨课，没有实验。据目前所知，这两种课都不需要上。

* KP
1. 该课程需要给老师发送邮件商定你的KP题目，过程十分折磨。该课程什么都不需要干，写这个KP就行了。
2. 在十一月之前和他沟通会非常艰难，因为这时候他会刁难你，压力你，因为他期待你能写出一篇真正能发表的文章，顺便附带他的名字。
3. 在十一月沟通，他又会变得焦躁，嚷嚷着时间不多了。
4. 在十二月沟通，额，他会几乎拒绝沟通，让你随便定一个主题，尽快开始。
5. 主题从他给出的列表中选取，PPT中有对KP的要求，列表和PPT几乎是不变的。我们一般选择方向2.
6. 该老师会出尔反尔，据基里洛夫报道，存在前一天说通过，第二天来找他又说不给通过的现象.我们有一位同学，从11月开始与他沟通，他很愉快地同意了她的题目，经过几轮修改之后，又拒绝了他的题目，他是我的好朋友，老师说我们题目相似，事实上，我无话可说，我并不觉得相似，我们期间也没有任何交流。
7. 我从九月与他沟通，受尽了折磨。在12月29日回复邮件通过了KP，于30日完成了KP封面的签字。但是还不确定我是否通过KP，因为还没签本。
* 考试
2026/1/12 我现在正在参加他的考试答疑，只有我一个中国人来了，老头在人群中指出了我，说道：你是唯一一个提交报告的外国人，你会得到3或者4分。现在我觉得参加演讲课还是很有意义的，至少能让他认识你，我希望这会增加通过考试和KP的机会，回见。
2026/1/13 一位不幸的战士倒下了 我觉得是直接和老师用翻译器交流让老头觉得不礼貌 老头很傲娇 挂科一位；一个四分，其他人是三分。
2026/1/15 我获得了4分 其他人为3分。我的KP也顺利签了--目前所有外国人只有我通过了，四分，运气好罢了。考试是如何进行的：开始时我们拿到了题签，然后约写了三个小时，没有指定考试结束时间。我写了两个正反面，相关知识都写上了，很顺利地通过了答辩，险些获得5分，有一个问题我写了但是没想起来。同场考试的同学表示，老师见人下菜碟，他问的我的问题很简单，我很轻松地回答，但是我的朋友们被问到了复杂公式里的什么参数是什么东西，这些信息不存在于PPT中，也许只是想给3分的理由罢了。
其余同学将在下学期继续与他沟通KP。
