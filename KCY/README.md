## 学科共有6个实验，3个摘要，1个作业
    1. 建议使用现有的文件，而不是自己写。(如果你严格按照他的日程表，每周去答辩的话可以考虑自己写，不过还是建议用现成的，我认为他不关心是否抄袭，只关心格式对错，曾经存在前后几个人连续用同一份报告答辩，且存在相同问题，老师并没有意见)
    2. 他会在群里发布摘要的登记表，建议尽早填写，选择可以直接提交的题目。相信我，已有的材料比自己写的要更好。(因为已经根据他的要求改过了)。
    3. 实验和摘要都需要在答辩当天发到邮箱，在实验课上点名上去答辩，他把文件拖入他的库里，就算答辩通过。期间不询问任何问题(即使有，装作听不懂)。他会告诉你该改些什么。
    4. 作业为PPT汇报，在学期末进行，小组合作。
    5. 建议使用ChatGPT而不是DEEPSEEK。PPT请使用KIMI生成并手动修改。
    6. 考试的答疑会被用来答辩实验，这次很水，但是你不能在这次答所有的实验。这意味着，你可以在日常答辩几次，但不必担心他会卡你的实验。这次答辩我没有参加，因为我早就全过了^oo^
    7. 考试使用word作答，像答辩实验一样发送邮箱然后排队，据我所知不会为难人。
